# Issue #7: テキスト内容比較（行単位）

## 目的
`different` に分類されたファイルの中身を行単位で比較表示できるようにする。

## スコープ
- `compare:file-diff` IPC を追加
- テキストファイルの行単位差分を返却
- バイナリ判定（拡張子 + null byte）
- サイズ上限を超えるファイルの比較回避

## 受け入れ条件
- [ ] `different` ファイルを選択すると差分を取得できる
- [ ] テキストは行単位差分が表示される
- [ ] バイナリはMVP対象外メッセージが出る
- [ ] `npm run lint` / `npm run build` が通る
