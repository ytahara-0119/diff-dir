# Issue #4: ディレクトリ再帰走査モジュール

## 目的
比較処理の前段として、指定フォルダ配下のファイル情報を再帰的に収集するモジュールを実装する。

## スコープ
- `fs/promises` を使った再帰走査
- 相対パスの収集
- メタ情報（`size`, `mtimeMs`）取得
- シンボリックリンクはスキップ

## 受け入れ条件
- [ ] 指定フォルダ配下の全ファイル情報を配列で取得できる
- [ ] 相対パスは `/` 区切りで正規化される
- [ ] ソート済みで返却される
- [ ] `npm run lint` / `npm run build` が通る
