# Issue #5: 除外ルール実装（.git / node_modules など）

## 目的
比較対象のノイズを減らし、走査性能を安定化するために除外ルールを導入する。

## スコープ
- デフォルト除外名: `.git`, `node_modules`
- カスタム除外名: IPCリクエストから任意の除外名を受け取れる
- 走査時にディレクトリ/ファイル名一致で除外

## 受け入れ条件
- [ ] デフォルト除外が走査結果に含まれない
- [ ] カスタム除外名を設定できる
- [ ] デフォルト + カスタムが重複なく適用される
- [ ] `npm run lint` / `npm run build` が通る
