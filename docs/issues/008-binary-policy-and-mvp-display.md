# Issue #8: バイナリ判定とMVP表示の整理

## 目的
バイナリ判定・サイズ上限・表示仕様を統一し、比較結果の見通しを改善する。

## スコープ
- バイナリ拡張子とテキスト差分サイズ上限を共通設定化
- 比較一覧に差分ヒント（text / binary / too_large）を表示
- `binary` / `too_large` ではファイル読込を行わずMVPメッセージを表示
- File diff エラー文言の統一

## 受け入れ条件
- [ ] 判定ルールが1箇所で管理される
- [ ] 一覧で差分可否がクリック前に分かる
- [ ] `binary` / `too_large` 表示が一貫する
- [ ] `npm run lint` / `npm run build` が通る
