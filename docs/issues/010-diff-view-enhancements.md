# Issue #10: 差分ビュー画面の強化

## 目的
差分内容を読みやすくし、失敗時にも再操作しやすい差分ビューへ改善する。

## スコープ
- 差分行タイプ（added/removed/context）の色分け
- 差分行の折り返し切替
- 長い context run の省略表示と展開切替
- 選択ファイルのメタ情報表示（size/mtime）
- 差分取得失敗時の再試行ボタン

## 受け入れ条件
- [ ] 差分行タイプが見分けられる
- [ ] wrap ON/OFF を切替できる
- [ ] 長い context を省略/展開できる
- [ ] 失敗時にリトライできる
- [ ] `npm run lint` / `npm run build` が通る
